# Projeto Anna Bella

Projeto desenvolvido durante meus estudos de HTML5 e CSS3 no curso **Desenvolvimento Web Completo** da Udemy.

## Objetivo

Praticar os conceitos fundamentais de desenvolvimento web, incluindo:

- Estruturação de páginas com HTML5
- Navegação entre páginas
- Organização de arquivos
- Estilização com CSS3
- Aplicação de cores, fontes e layouts

## Tecnologias utilizadas

- HTML5
- CSS3

## Funcionalidades

- Página inicial
- Biografia
- Campanhas publicitárias
- Página de contato
- Navegação entre páginas

## Aprendizados

Durante este projeto pude reforçar conceitos importantes como:

- Estrutura semântica do HTML
- Separação entre estrutura e estilo
- Organização de arquivos
- Utilização de seletores CSS
- Estilização de menus e conteúdos
